# Прогноз стоимости подержанных автомобилей

Проект по курсу Машинное обучение HW1_part2
Цель: построить регрессионную модель для предсказания цены подержанного автомобиля на основе его характеристик и развернуть её в интерактивном Streamlit-приложении.
Выполнил: Студент 1-го курса магистратуры ИИ25 Иванов Дмитрий

Ссылка на сервис: https://hw1part2ivanovda.streamlit.app/
---

## Структура репозитория

- `HW1_Regression_with_inference_base_IvanovD.ipynb` — полный ноутбук с EDA, препроцессингом, обучением и оценкой моделей из HW1_part1.
- `model.pkl` — сохранённая модель Ridge и все необходимые препроцессоры (`TargetEncoder`, `OneHotEncoder`, `StandardScgitaler`).
- `app.py` — Streamlit-приложение для интерактивного предсказания.
- `data/df_train_processed.csv` — очищенный тренировочный датасет (с `log_selling_price`).
- `report.md` — подробный отчёт с выводами и оценкой сервиса.
- `requirements.txt` — зависимости для воспроизводимости.

---

## Запуск приложения

1. Установите зависимости:
   ```bash
    pip install -r requirements.txt
2. Запустите Streamlit:
    streamlit run app.py

## Выводы и оценка сервиса

1. Проведён разведочный анализ данных - EDA
2. Проведена обработка данных, предобработаны и числовые и категориальные и текстовый признаки
3. Опробовано несколько линйных моделей с усложнениями в виде регуляризации
4. Подобраны гиперпараметры для лучшей модели - Ridge-регрессия (alpha=0.1)
5. Достигнут хороший результат по метрике R² ≈ 0.897
6. Бизнес-метрика бюыла решена неправильно. Не стал исправлять,что бы дурость была видна.
7. На сервис в стримлите добавлена итоговая модель, EDA на предобработанном дата-сете и возможность попрогнозировать цены для роапзных машин
8. Добавлена визуализация коэффициентов итоговой модели:
    Самые важные признаки, увеличивающие цену - машина из тест-драйва в салоне, год выпуска, мощность двигателя и марка и модель;
    Самые важные метрики, уменьшающие цену - многоместный кузов, ручная коробка и большое количество предыдущих хозяев.
